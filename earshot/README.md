# Earshot

This directory holds the project's components — each with its own README for
setup, tuning, and troubleshooting:

- [`ml/`](ml/) — offline audio-event detection (YAMNet, trained alarm head,
  teach mode)
- [`backend/`](backend/) — FastAPI switchboard: one event in, four alerts out
- [`frontend/`](frontend/) — the dashboard and the wrist wearable
- [`pi/`](pi/) — the physical alert unit (LED, motor, buzzer)

Start with the **[repository README](../README.md)** for the pitch,
architecture, quick start, and team.
